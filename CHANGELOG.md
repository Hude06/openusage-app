# Changelog

## v1.0.0

Initial release.

- Real-time Claude and Codex token usage monitoring
- Session (5hr) and weekly (7d) window tracking with countdown timers
- Segmented progress bars with status color coding
- Claude model breakdown (Opus/Sonnet) with compact bars
- Codex credits remaining and daily spend
- 24-hour usage sparklines from local SQLite history
- Configurable polling interval (30s - 5m)
- OAuth token management with automatic refresh
- Nothing Design System UI (OLED black, Space Grotesk/Mono/Doto typography)
- macOS native integration (hidden titlebar, dark mode)
